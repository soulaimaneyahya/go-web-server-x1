## Installation

- Clone the repository:

```shell
git clone https://github.com/soulaimaneyahya/go-web-server-x1.git
```

### Usage

1. Compile go-web-server-x1:

```sh
go mod init go-web-server-x1 && go mod tidy
```

```sh
go fmt ./src/*.go
```

```sh
make build
```

```sh
make run
```

```sh
make clean
```

## Posts

```sh
http://localhost:8080/posts
```
